## 🎯 テストの仕方

### 1. ゲーム起動
```bash
# ブラウザでアクセス
http://localhost:5173

# ログイン（テストユーザー）
username: testuser2
password: password123
```

### 2. ゲームプレイ
1. メインメニューで時間モードを選択（1分、5分など）
2. ゲーム画面に移動
3. WASDまたは矢印キーでプレイヤーを移動
4. タイマーが0になるまで待つか、ESCでポーズ

### 3. 確認項目
- ✅ タイマーが正確にカウントダウンされるか
- ✅ プレイヤーがスムーズに移動するか
- ✅ ポーズ機能が動作するか（タイマーは止まらない）
- ✅ タイムアップでゲームオーバー画面が表示されるか
- ✅ ESCキーでメニューに戻れるか

## 🔧 技術的な詳細

### アーキテクチャ
```
GameEngine
  ├── GameStateManager (状態管理)
  ├── TimerSystem (タイマー)
  ├── Player (プレイヤー)
  └── render() (描画)
```

### ゲームループ
```typescript
gameLoop() {
  deltaTime計算
  update(deltaTime) {
    タイマー更新
    プレイヤー更新
  }
  render() {
    背景描画
    プレイヤー描画
    UI描画
  }
  requestAnimationFrame(gameLoop)
}
```

### 入力処理
- キーボードイベントをSetで管理
- 毎フレームSetをチェックして移動ベクトル計算
- 正規化して一定速度で移動
