## 🎯 エフェクトシステムの特徴

### アーキテクチャ
```typescript
EffectSystem
├── Effect[] - エフェクトの配列
├── update(deltaTime) - エフェクトの更新
├── render(ctx, camera) - エフェクトの描画
└── addXxxEffect() - 各種エフェクトの追加メソッド
```

### エフェクトタイプ
```typescript
enum EffectType {
  DAMAGE_TEXT,    // ダメージ数値
  HEAL_TEXT,      // 回復数値
  LEVEL_UP,       // レベルアップ
  HIT_FLASH,      // ヒットフラッシュ
  SKILL_AURA,     // スキルオーラ
  ITEM_PICKUP,    // アイテムピックアップ
  EXPLOSION,      // 爆発
}
```

### エフェクトデータ構造
```typescript
interface Effect {
  id: string              // ユニークID
  type: EffectType        // エフェクトタイプ
  position: Vector2       // 位置
  lifetime: number        // 残り時間
  maxLifetime: number     // 最大時間
  data?: any              // エフェクト固有データ
}
```

## 🎮 統合ポイント

### GameEngineとの統合
1. **初期化**
   ```typescript
   this.effectSystem = new EffectSystem()
   ```

2. **更新ループ**
   ```typescript
   this.effectSystem.update(deltaTime)
   ```

3. **描画ループ**
   ```typescript
   this.effectSystem.render(ctx, cameraX, cameraY)
   ```

### 戦闘システムとの統合
```typescript
// ダメージエフェクト
this.effectSystem.addDamageText(position, damage)
this.effectSystem.addHitFlash(position, isPlayer)

// レベルアップエフェクト
if (leveledUp) {
  this.effectSystem.addLevelUpEffect(playerPos)
}
```

### アイテムシステムとの統合
```typescript
// アイテムピックアップ
this.effectSystem.addItemPickup(playerPos, itemName, color)

// アイテム使用
if (type === ItemType.HEALTH_POTION) {
  this.effectSystem.addHealText(playerPos, 50)
  this.effectSystem.addSkillAura(playerPos, '#22c55e', 0.5)
} else if (type === ItemType.BOMB) {
  this.effectSystem.addExplosion(playerPos, 100)
}
```

### スキルシステムとの統合
```typescript
// プレイヤーからのコールバック
this.player.onSkillUsed = (skillType: string) => {
  this.handleSkillUsed(skillType)
}

// スキルエフェクトの表示
this.effectSystem.addSkillAura(playerPos, color, duration)
```
